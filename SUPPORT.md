## Перед обращением в поддержку

Чтобы мы могли помочь быстрее, проверьте сами:

1. **Используете ли вы последнюю версию плагина?**  
   Проверить можно в WordPress: **Плагины → Доступные обновления**, либо в [Releases](https://github.com/RuCoder-sudo/wp-ru-max/releases). Многие проблемы уже исправлены.

2. **Прочитали ли вы [Wiki](https://github.com/RuCoder-sudo/wp-ru-max/wiki) и FAQ?**  
   Самые частые вопросы (например, «почему звук не звучит сразу при загрузке страницы») разобраны в FAQ внутри `readme.txt` и в Wiki.

3. **Поискали ли среди [Issues](https://github.com/RuCoder-sudo/wp-ru-max/issues?q=is%3Aissue)?**  
   Включите поиск по закрытым Issues — возможно, проблема уже решена.

4. **Включён ли режим отладки WordPress?**  
   Если плагин работает не так, как ожидается, добавьте в `wp-config.php`:
   ```php
   define( 'WP_DEBUG', true );
   define( 'WP_DEBUG_LOG', true );
   define( 'WP_DEBUG_DISPLAY', false );
   ```
   Логи появятся в `wp-content/debug.log` — приложите их к Issue.

## Что приложить к запросу в поддержку

Чтобы получить ответ быстрее, сразу укажите:

- **Версия плагина WP Ru-max** (см. в админке: Плагины)
- **Версия WordPress** (см. в нижнем правом углу админки)
- **Версия PHP** (Инструменты → Здоровье сайта → Информация → Сервер)
- **Тема и активные плагины** — особенно если конфликт
- **Шаги для воспроизведения проблемы**
- **Скриншоты или видео** при UI-проблемах
- **Логи** из `wp-content/debug.log` или вкладки «Логи» плагина
- **Конкретное сообщение об ошибке** (без сокращений)

## Сроки ответа

| Канал | Ожидаемое время ответа |
|---|---|
| GitHub Issues (баги) | 1–3 рабочих дня |
| GitHub Discussions (вопросы) | 1–5 рабочих дней |
| Email `support@рукодер.рф` (лицензии) | 1–2 рабочих дня |
| Telegram (срочные случаи) | в течение нескольких часов в рабочее время |
| Уязвимости безопасности | подтверждение в течение 72 часов (см. [SECURITY.md](SECURITY.md)) |

Сроки указаны для рабочих дней по Московскому времени. В выходные и праздники ответ может занять дольше.

Для пользователей с активной лицензией доступна **приоритетная поддержка** через email `support@рукодер.рф` и Telegram. Подробности — на [рукодер.рф](https://рукодер.рф).

- Консультации по интеграции WordPress-сайта с MAX

Запрос на коммерческие услуги: `support@рукодер.рф`.
