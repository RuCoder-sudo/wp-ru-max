Интеграция WordPress с мессенджером MAX (max.ru). Автопубликация записей, уведомления с форм и чат-виджет для сайта.

== Description ==

WP Ru-max — плагин для интеграции вашего WordPress сайта с российским мессенджером MAX (max.ru). Версия 1.0.34.

**Три модуля в одном плагине:**

* Автопубликация — новые и обновлённые записи/страницы автоматически отправляются в канал MAX с миниатюрой и ссылкой «Читать полностью».
* Личные уведомления — перехват wp_mail() — все письма WordPress (заказы WooCommerce, заявки с форм CF7, Elementor, Gravity Forms, Ninja Forms и других) дублируются в личный чат с ботом MAX.
* Чат-виджет — плавающая кнопка MAX с приветствием, анимацией «печатания», звуком, задержкой появления и попапом удержания.

**Новое в версии 1.0.34:**

* WooCommerce — фильтр уведомлений по статусам заказа (ожидает оплаты, в обработке, выполнен, отменён и др.).
* Защита от дублей WooCommerce — одно уведомление о заказе больше не приходит 2–3 раза подряд.
* Настройки фильтра в разделе «Личные уведомления» → «WooCommerce — фильтр заказов».

**Поддерживаемые плагины уведомлений (стандарт wp_mail()):**

* WooCommerce (новые заказы, смена статуса, оплата)
* Contact Form 7
* Elementor Forms (включая Pro)
* Gravity Forms
* Ninja Forms
* WP Mail SMTP / FluentSMTP / Postman SMTP
* Jetpack
* Любые плагины, использующие стандартный wp_mail() WordPress

**Важно:** плагин работает только со стандартом WordPress wp_mail(). Плагины, использующие нестандартные механизмы отправки почты в обход wp_mail(), не поддерживаются (это ограничение таких плагинов, а не нашего).

**Совместимость:** WordPress 5.8 – 6.7 · PHP 7.4 – 8.3 · WooCommerce · WordPress Multisite


== Installation ==

1. Загрузите папку `wp-ru-max` в директорию `/wp-content/plugins/`
2. Активируйте плагин через меню «Плагины» в WordPress
3. Перейдите в раздел **Ru-max** в меню администратора
4. Введите токен бота MAX на вкладке «Главная»
5. Проверьте подключение и настройте нужные модули

== Frequently Asked Questions ==

= Где получить токен бота? =

Токен находится на платформе MAX для партнёров (https://max.ru/partner) в разделе «Чат-боты» → «Интеграция» → «Получить токен».

= Как узнать ID моего канала? =

Для публичного канала используйте его никнейм в формате @channel_name. Для групп используйте числовой ID.

= Работает ли плагин с WooCommerce? =

Да! Плагин автоматически перехватывает все email-уведомления WooCommerce о новых заказах и пересылает их в MAX.

= Почему звук в виджете не срабатывает сразу при загрузке страницы? =

Браузеры запрещают воспроизводить звук без взаимодействия пользователя со страницей. Звук воспроизводится при первом касании, клике или прокрутке страницы пользователем.
