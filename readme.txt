Интеграция WordPress с мессенджером MAX (max.ru). Автопубликация записей, уведомления с форм и чат-виджет для сайта.

== Description ==

WP Ru-max — плагин для интеграции вашего WordPress сайта с российским мессенджером MAX (max.ru).

**Возможности:**

* Автопубликация — автоматически отправляет новые и обновлённые записи/страницы в канал MAX
* Личные уведомления — перехватывает email-уведомления WordPress и пересылает их в MAX (заявки с форм WooCommerce, Contact Form 7, Elementor и любых других)
* Встроенные кнопки — добавляйте кнопки с URL под публикациями и уведомлениями в MAX
* Шаблоны сообщений — настраивайте шаблон публикации с переменными {title}, {excerpt}, {url}, {meta_FIELD}, {acf_FIELD} и другими
* Чат-виджет — добавляет плавающую кнопку MAX на сайт с анимацией печатания приветственного сообщения, задержкой появления, звуковым уведомлением и анимацией привлечения внимания
* История и логи — полная история всех событий с возможностью фильтрации
* Тестирование — встроенная функция проверки подключения и отправки тестовых сообщений

**Чат-виджет — расширенные настройки (v1.0.16):**

* Отступ снизу — задайте точное расстояние значка от нижнего края экрана (0–200 px) с помощью ползунка
* Задержка появления — выберите, когда показать виджет: сразу, через 5, 8, 10 или 15 секунд после загрузки страницы
* Звук уведомления — три варианта звука на выбор (Новое сообщение, Всплывающее окно, Тихий сигнал) с кнопкой предварительного прослушивания в настройках; выбор момента воспроизведения: через 3, 6 или 9 секунд после появления виджета
* Анимация привлечения внимания — Пульсация, Рябь (расходящиеся круги), Подпрыгивание, Покачивание — запускается автоматически после исчезновения приветственного пузыря
* Кнопка закрытия — посетитель может закрыть приветственный пузырь крестиком

**Поддерживаемые плагины для форм:**

* WooCommerce (новые заказы)
* Contact Form 7
* Elementor Forms
* Gravity Forms
* Ninja Forms
* Любые другие плагины, использующие стандартный WordPress mail

== Installation ==

1. Загрузите папку `wp-ru-max` в директорию `/wp-content/plugins/`
2. Активируйте плагин через меню «Плагины» в WordPress
3. Перейдите в раздел **Ru-max** в меню администратора
4. Введите токен бота MAX на вкладке «Главная»
5. Проверьте подключение и настройте нужные модули

== Frequently Asked Questions ==

= Где получить токен бота? =

Токен находится на платформе MAX для партнёров (https://max.ru/partner) в разделе «Чат-боты» → «Интеграция» → «Получить токен».

= Как узнать ID моего канала? =

Для публичного канала используйте его никнейм в формате @channel_name. Для групп используйте числовой ID.

= Работает ли плагин с WooCommerce? =

Да! Плагин автоматически перехватывает все email-уведомления WooCommerce о новых заказах и пересылает их в MAX.

= Почему звук в виджете не срабатывает сразу при загрузке страницы? =

Браузеры запрещают воспроизводить звук без взаимодействия пользователя со страницей. Звук воспроизводится при первом касании, клике или прокрутке страницы пользователем — сразу, как только оно происходит.
