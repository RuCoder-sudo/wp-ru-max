## О плагине

**WP Ru-max** подключает WordPress-сайт к мессенджеру [MAX (max.ru)](https://max.ru) и решает три задачи:

- **Автопубликация** - новые записи и страницы автоматически отправляются в канал MAX.
- **Уведомления** - все email WordPress (заказы WooCommerce, заявки с форм) дублируются в личный чат с ботом.
- **Чат-виджет** - настраиваемая плавающая кнопка MAX на сайте для связи с посетителями.

Лицензирование и управление ключами реализовано через REST API сайта [рукодер.рф](https://рукодер.рф).

## Возможности

### Автопубликация записей

| Функция | Описание |
|---|---|
| Отправка при публикации | Новая или обновлённая запись / страница → сообщение в канал MAX |
| Миниатюра | Изображение прикрепляется к сообщению (с лимитом размера) |
| Ссылка «Читать полностью» | Опционально — можно отключить |
| Ручная отправка | Кнопка «Отправить в MAX» прямо в редакторе записи |
| Типы записей | Любые публичные типы (posts, pages, CPT) |
| Отложенная отправка | Задержка 0 / 30 с / 1 / 2 / 5 мин - исключает гонку условий |
| Повторные попытки | Настраиваемое число попыток и интервал при ошибке API |
| Фильтр категорий/тегов | Отправка только из выбранных категорий или тегов |

### Уведомления в личный чат

| Функция | Описание |
|---|---|
| Перехват `wp_mail()` | Все email-уведомления WordPress → в MAX |
| WooCommerce | Новый заказ, смена статуса, оплата и т.д. |
| Contact Form 7 | Уведомления со всех форм |
| Elementor Forms | Включая Elementor Pro |
| Gravity Forms | Полная поддержка |

### Чат-виджет

| Функция | Описание |
|---|---|
| Плавающая кнопка | Иконка MAX в углу сайта |
| Приветственное сообщение | Текст с анимацией «печатания» |
| Анимации внимания | Пульс, рябь, подпрыгивание, встряска, свечение, вращение |
| Звуковое уведомление | 3 варианта (синтез через Web Audio API) |
| Задержка появления | 0 / 5 / 8 / 10 / 15 секунд |
| Отступ от края | Слайдер 0–200 px |

### Новое в v1.0.30

| Функция | Описание |
|---|---|
| Push-блок перемещён | Блок «Отправка push-уведомлений» теперь находится сразу под «Канал(ы)» |
| Уведомления обновления плагинов | Сообщение в MAX при обновлении плагинов или ядра WordPress |
| Уведомления ошибок сайта | Сообщение в MAX при критической PHP-ошибке (не чаще 1 раза в 5 мин) |
| Правила отправки в уведомлениях | Новый раздел в «Личных уведомлениях» — два независимых переключателя |

## В ближайших обновлениях

> Запланировано к реализаци

- **Передача событий в Яндекс.Метрику** — анализ поведения пользователей через события MAX-виджета
- **CRM + автоматизация** — интеграция с amoCRM и Bitrix24, автоматизация отдела продаж
- **Кнопка «Поделиться в MAX»** — позволит пользователям делиться контентом сайта в мессенджере MAX
- **Хэштеги и упоминания** — автоматические хэштеги в постах, упоминания менеджеров о важных событиях
- **MAX-авторизация (OAuth)** — вход на сайт через аккаунт MAX, автоподтягивание имени/email/фото в профиль WordPress

## Установка

### Способ 1: через панель WordPress

1. Скачайте ZIP-архив плагина
2. Перейдите в **Плагины → Добавить новый → Загрузить плагин**
3. Выберите файл `wp-ru-max.zip` и нажмите **Установить**
4. Нажмите **Активировать плагин**

### Способ 2: через FTP / SSH

1. Клонируйте репозиторий или распакуйте архив:
   ```bash
   git clone https://github.com/RuCoder-sudo/wp-ru-max.git
   ```
2. Загрузите папку `wp-ru-max` в `/wp-content/plugins/`
3. Активируйте плагин в **Плагины → Установленные плагины**

* Добавлено: раздел «Отправка push-уведомлений в MAX» во вкладке «Отправка публикаций».
* Добавлено: поддержка WP Mail SMTP / FluentSMTP / Postman SMTP — перехват писем на приоритете 5.
