Установка

  1. Скачайте ZIP-архив из Releases.
  2. В админке WordPress: Плагины → Добавить новый → Загрузить плагин.
  3. Активируйте плагин.

Активация лицензии

Перейдите в меню Ru-max → Активация. Введите ключ формата WPRM-XXXX-XXXX-XXXX-XXXX.

Нет ключа? Запросите через форму на той же вкладке или купите на Kwork.

Настройка бота MAX

Получите токен на max.ru/partner (Чат-боты → Интеграция → Получить токен). Вставьте токен в Ru-max → Главная и нажмите «Проверить подключение».

Автопубликация

В разделе Ru-max → Публикация укажите ID канала (например @my_channel) и включите отправку.

Уведомления

Включите перехват wp_mail() в разделе Ru-max → Уведомления. Все письма будут дублироваться в личный чат с ботом.

Чат-виджет

Настройте внешний вид, анимации, звук и задержку появления в разделе Ru-max → Виджет.

Логи и тестирование

Все отправки и ошибки сохраняются в таблицу логов (Ru-max → Логи). Там же можно посмотреть JSON запросов.
