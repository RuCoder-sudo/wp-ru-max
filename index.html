Возможности

Всё, что нужно для интеграции с MAX

Шесть модулей, которые работают «из коробки» — без лишних настроек и сторонних сервисов.

📝

Автопубликация

Новая или обновлённая запись/страница автоматически отправляется в канал MAX, с миниатюрой и опциональной ссылкой «Читать полностью».

📩

Личные уведомления

Перехват wp_mail() — все письма WordPress (заказы WooCommerce, заявки с форм) дублируются в личный чат с ботом MAX.

💬

Чат-виджет

Плавающая кнопка MAX с приветствием, анимацией «печатания», звуком, задержкой появления и попапом удержания.

📊

История и логи

Полная таблица событий: все отправки, ошибки, тесты. Фильтрация по типу/дате/статусу и просмотр JSON-лога каждого запроса.

🧪

Тестирование

Встроенная проверка соединения с ботом MAX и отправка тестовых сообщений в любой канал или чат — прямо из админки.

🔐

Лицензирование

Привязка к домену через REST API сайта рукодер.рф. Часовая фоновая перепроверка ключа и кнопка «Проверить лицензию сейчас».

Установка

Запуск за 5 минут

Стандартная установка плагина WordPress + ввод лицензионного ключа и токена бота.

Скачайте архив

Возьмите ZIP последней версии в GitHub Releases.

Загрузите в WordPress

Плагины → Добавить новый → Загрузить плагин → выберите wp-ru-max.zip → Установить.

Активируйте лицензию

Меню Ru-max → Активация. Ключ имеет формат WPRM-XXXX-XXXX-XXXX-XXXX.

Подключите бота MAX

Получите токен на max.ru/partner и вставьте на вкладке «Главная».

# Через FTP/SSH
git clone https://github.com/RuCoder-sudo/wp-ru-max.git \
    /var/www/site/wp-content/plugins/wp-ru-max

Вопросы и ответы

Коротко о главном

Где взять токен бота MAX?

На платформе max.ru/partner → раздел «Чат-боты» → «Интеграция» → «Получить токен». Вставьте его на вкладке Ru-max → Главная и нажмите «Проверить подключение».

Как узнать ID канала или группы?

Для публичного канала — никнейм с @, например @my_channel. Для группы — числовой ID, который можно получить через бота @get_id_bot в самом мессенджере MAX.

Работает ли плагин с WooCommerce?

Да. Все email-уведомления WooCommerce (новый заказ, смена статуса, оплата и т.д.) перехватываются и пересылаются в личный чат с ботом MAX.

Поддерживаются ли Contact Form 7, Elementor, Gravity Forms?

Да. Плагин работает с любыми формами, отправляющими уведомления через стандартный wp_mail() WordPress: CF7, Elementor Forms, Gravity Forms, Ninja Forms и другими.

Лицензия действительно бессрочная?

Да. Один платёж — пожизненная активация для указанного домена. Никаких подписок, продлений и абонентской платы. Все обновления плагина включены в стоимость.

Что будет, если ключ отозвать?

Плагин ходит на сервер рукодер.рф и сверяет статус ключа: при открытии вкладки «Активация» — мгновенно, в фоне — раз в час. Если ключ отозван, статус сразу переключается на suspended и платные функции отключаются.

Как работают обновления?

Плагин раз в 12 часов проверяет GitHub Releases. Если есть новая версия — в WordPress появится стандартное уведомление «Доступно обновление», устанавливается в один клик.

Почему звук виджета не играет сразу?

Браузеры блокируют автовоспроизведение звука до первого взаимодействия пользователя со страницей. Звук сыграет сразу после первого клика, касания или прокрутки.
