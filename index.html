🚀 WP Ru-max

Интеграция WordPress с мессенджером MAX (max.ru) — автопубликация записей, пересылка уведомлений WooCommerce / CF7 / Elementor и настраиваемый чат-виджет с анимацией и звуком.

GitHub Купить лицензию (Kwork)

✨ Основные возможности

  • Автопубликация — новые записи и страницы автоматически отправляются в канал MAX
  • Уведомления — все email WordPress (заказы WooCommerce, заявки с форм) дублируются в личный чат с ботом
  • Чат-виджет — настраиваемая плавающая кнопка MAX на сайте с анимацией и звуком
  • История и логи — полная таблица всех событий с JSON-логами
  • Бессрочная лицензия — платите один раз, пользуетесь навсегда

📦 Быстрая установка

  1. Скачайте ZIP-архив плагина с GitHub Releases
  2. В админке WordPress перейдите в Плагины → Добавить новый → Загрузить плагин
  3. Выберите файл wp-ru-max.zip и нажмите Установить
  4. После установки нажмите Активировать
  5. Перейдите в меню Ru-max → Активация и введите лицензионный ключ
